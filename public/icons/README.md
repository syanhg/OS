# Drop your icon assets here

Any PNG placed in this folder automatically replaces the built-in
placeholder artwork. Use square images, ideally 128×128 or larger.

| File name          | Used for                          |
| ------------------ | --------------------------------- |
| `finder.png`       | Finder (dock, windows)            |
| `dashboard.png`    | Dashboard (dock)                  |
| `ichat.png`        | iChat (dock)                      |
| `ie.png`           | Internet Explorer (dock)          |
| `textedit.png`     | TextEdit (dock, Finder files)     |
| `appletstore.png`  | Applet Store (desktop, dock)      |
| `trash.png`        | Trash (dock)                      |
| `hd.png`           | Macintosh HD (desktop)            |
| `ipod.png`         | iPod (desktop)                    |
| `folder.png`       | Folders inside Finder             |
| `apple.png`        | (reserved) Apple logo artwork     |

Wallpaper: put a `wallpaper.jpg` directly in `public/` (next to this
folder's parent) and it replaces the default Aqua-blue gradient.
